## command location & version

### which
Locate where a command is stored
```bash
which <command>
```
Examples
```bash
which git
which node
```
### [-v] 
Find the version of the command
```bash
<command> -v
```
Examples
```bash
git -v
node -v
```
### [--version] 
If -v doesn't show a version try:
```bash
<command> --version
```
Examples
```bash
bash --version
```
